# Лабораторная работа №5 — WpfContacts

**Дисциплина:** Проектирование программного интерфейса  
**Вариант 16**

WPF-приложение на C#, реализующее функционал **Телефонной книги**. В работе используются элементы компоновки WPF (Grid, Expander, ScrollViewer), компонент DataGrid для отображения данных, а также механизм сериализации объектов в XML-файл с помощью XmlSerializer.

## Что реализовано

- **CRUD-операции** с контактами: добавление, редактирование, удаление и полная очистка таблицы.
- **Персистентное хранение данных**: сохранение и загрузка контактов в локальный **XML-файл** (`contacts.xml`) через сериализацию объектов.
- **Синхронизация полей**: при выборе строки в DataGrid данные контакта автоматически заполняют поля ввода; после удаления последней строки поля очищаются.
- **Компоновка интерфейса**: использование Expander для группировки элементов (форма ввода и панель управления), ScrollViewer для прокрутки содержимого DataGrid.
- **Интерфейс**: окно по центру экрана, минимальный размер, без кнопки «Развернуть».
- **Чистый код**, осмысленные имена, комментарии на русском языке.
